## Shopping Cart System

### Description
This is a console-based Shopping Cart System developed using Java. It allows users to manage products through a simple menu-driven application.

### Features
- Add Product
- View Products
- Remove Product
- Search Product
- Display Total Number of Products

### Technologies Used
- Java
- ArrayList
- Scanner
- Object-Oriented Programming (Basic)
